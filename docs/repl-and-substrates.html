🏁 Conclusion — REPL experiment closed

Bottom line

A full Harbor / Terminal-Bench 2 sweep settled the question the REPL experiment set out to answer. On TB2, a repl-centric tool selection is near as good as the standard tool set — perhaps a little worse, and very likely not better. There is no measured upside that justifies continued investment, so we are treating the REPL experiment as done. If new ideas about a REPL or alternative substrate surface later, we reopen the topic freshly rather than carrying this plan forward.

Headline numbers from the 89-task sweep (Sonnet-medium, claude-sonnet-4-6, omega v0.1.15 + v0.1.16 correction):

  • repl-centric: 52 / 89 ≈ 58.4 % (as-run 50 / 89; two tasks re-run after omega-cli bug fixes).
  • standard baseline: 53 / 89 ≈ 59.6 %.
  • Delta: roughly −1 to −3 pp — within plausible single-trial noise, with no signal pointing to a REPL advantage.

The supporting sweep data, per-task error analysis, and the full experiment history (findings F-A through F-G, the substrate cost/fluency framing, the comparative agent survey, and the v0.1.8 → v0.1.13 hardening progression) are preserved in git history and in bench/README.md / bench/results/sweeps.json. They were removed from this document when the experiment was closed; recover them from version control if a future reopening needs them.

🧭 Decisions on record

Two decisions from this work outlive the experiment and remain in force.

  1. Q1 — Tool selection is start-only. Captured in SessionStartedEvent, immutable for the lifetime of the session. Caching hygiene; symmetry with model/effort/system-prompt; avoids growing the domain-event surface. Users change toolset by starting a new session.
  2. Strict session resume — dropped (with code deleted). Full reasoning at session-design.html §decision-drop-strict-resume. Short form: the observational-equivalence promise never survived world drift across resumes; the user's empirical usage data does not motivate the complexity; AI-based forensics over events.jsonl covers the real use case.
